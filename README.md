# ClaimFlow

ClaimFlow is a full stack claims management application built with ASP.NET Core Web API, React, and SQL Server.

## Tech Stack
- ASP.NET Core Web API
- React + Vite
- SQL Server
- Entity Framework Core
- JWT Authentication
- Recharts

## Project Structure
- backend/ClaimFlow.API
- frontend/claimflow-ui
- docs/

## Goal
To build a claims processing and tracking system with authentication, dashboards, error tracking, and state-based validation.
